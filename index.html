Try it right away!

You aren't required to use Javascript when building an app with boosted-html. You can manage all of your application with a set of attributes
boosted-html weights only 3kb (when not compressed). No need to worry about network performance
boosted-html offers you a component architecture with slots and attribute props to promote a DRY codebase and efficent code reduction
No need to store all of your HTML in one file. You can fetch external HTML easily thanks to boosted-html and start building with it right away
You can scope elements automatically, so that the element CSS doesn't interact with the body and viceversa
There's no build step: every change you make to the document is automatically updated in the browser

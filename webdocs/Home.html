Getting started


Create a new HTML file with the following content:

 <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <script src="https://andrealicheri.github.io/boosted-html/boostedhtml.min.js"></script>
    <title>Hello, world!</title>
</head>
<body>
    <template id="hello-world">
        <p>Hello {{prop:name}}</p>
    </template>
    <div b-read="hello-world" name="Andrea"></div>
</body>
</html>

If the output is "Hello, Andrea", congrats: you are set off to start using boosted-html
